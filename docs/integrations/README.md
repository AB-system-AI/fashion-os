# FashionOS Enterprise Integrations

Phase 15 delivers connectors, webhooks, API keys, communication channels, import/export jobs, OAuth, printers, and health monitoring — offline-first with sync to Supabase.

## Quick start

1. Bootstrap registers `integrationsModuleInitializerProvider` (8 sync processors + cross-module hooks).
2. Navigate to `/integrations` from Foundation.
3. Configure connectors, webhooks, and communication providers.
