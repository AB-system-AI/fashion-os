# FashionOS Enterprise Automation

Phase 14 delivers rules, workflows, scheduling, approvals, document templates, and AI-assisted suggestions — offline-first with sync across all automation entities.

## Quick start

1. Bootstrap registers `automationModuleInitializerProvider` (11 sync processors).
2. Navigate to `/automation` from Foundation.
3. Workflow: Rule → Workflow → Schedule → Execute → Log.
