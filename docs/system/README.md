# FashionOS Enterprise System Administration

Phase 16 delivers system administration, security center, monitoring, audit explorer, feature flags, maintenance mode, and diagnostics — offline-first with tenant-scoped sync.

## Quick start

1. Bootstrap registers `systemModuleInitializerProvider` (19 sync processors).
2. Navigate to `/system` from Foundation.
3. Requires `system.view` for dashboard; granular permissions per area.
