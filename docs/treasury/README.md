# FashionOS Enterprise Treasury

Phase 14 delivers cash management, bank accounts, transfers, payments, receipts, expenses, cheques, reconciliation, and cash forecasting — offline-first with integrations across Accounting, CRM, Purchasing, Sales OMS, HR, and Analytics.

## Quick start

1. Bootstrap registers `treasuryModuleInitializerProvider` (12 sync processors + integration).
2. Navigate to `/treasury` from Foundation.
3. Workflow: Cash/Bank setup → Transfers → Payments/Receipts → Reconciliation → Forecast.
