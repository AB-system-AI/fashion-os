# FashionOS Enterprise Sales OMS

Phase 13 delivers quotations, sales orders, shipments, deliveries, returns, exchanges, and CRM timeline — offline-first with integrations across Inventory, CRM, Accounting, Manufacturing, and Analytics.

## Quick start

1. Bootstrap registers `salesModuleInitializerProvider` (14 sync processors + integration).
2. Navigate to `/sales` from Foundation.
3. Workflow: Quotation → Sales Order → Reserve → Ship → Deliver.
