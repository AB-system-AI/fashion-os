# FashionOS Enterprise Assets & Maintenance

Phase 15 delivers fixed-asset register, depreciation, transfers, disposal, maintenance scheduling, contracts/warranties, and utilization reporting — offline-first with sync across all asset entities.

## Quick start

1. Bootstrap registers `assetsModuleInitializerProvider` (14 sync processors).
2. Navigate to `/assets` from Foundation.
3. Workflow: Register → Depreciate → Maintain → Transfer/Dispose.
